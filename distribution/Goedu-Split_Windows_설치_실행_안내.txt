Goedu-Split Windows 설치 및 실행 안내

1. 설치 파일로 실행하기

가장 쉬운 방법은 Goedu-Split-1.0.1-windows-setup.exe 파일을 사용하는 것입니다.

1) Windows 컴퓨터에서 자료 폴더를 엽니다.
2) Goedu-Split-1.0.1-windows-setup.exe 를 더블클릭합니다.
3) Windows 보안 경고 또는 SmartScreen 창이 뜨면 "추가 정보"를 누른 뒤 "실행"을 누릅니다.
4) 설치 화면 안내에 따라 설치합니다.
5) 설치가 끝나면 시작 메뉴 또는 바탕 화면 바로가기에서 Goedu-Split을 실행합니다.

2. 압축 파일로 실행하기

설치가 막히는 학교 PC에서는 Goedu-Split-1.0.1-windows.zip 을 사용할 수 있습니다.

1) Goedu-Split-1.0.1-windows.zip 을 오른쪽 클릭합니다.
2) "모두 압축 풀기"를 선택합니다.
3) 압축을 푼 폴더 안으로 들어갑니다.
4) Goedu-Split.exe 를 더블클릭합니다.

주의: zip 방식은 Goedu-Split.exe 파일만 따로 빼서 실행하면 안 됩니다. 압축을 푼 폴더 전체가 함께 있어야 합니다.

3. 처음 실행할 때 확인할 것

- 앱이 열리면 상단에 "성취평가 결과 분석" 화면이 보이면 정상입니다.
- 화면 오른쪽 위의 확대/축소 버튼이 50%까지 조절되는지 확인합니다.
- "AI 문항 검토" 탭에서 문항 자료와 성취기준·수준 자료를 불러올 수 있는지 확인합니다.
- "AI로 보강"은 로컬 MLX 또는 Ollama 서버가 켜져 있을 때 사용합니다. 서버가 없으면 로컬 초안만 사용할 수 있습니다.

4. 문제가 생겼을 때

- 실행이 차단되면 SmartScreen에서 "추가 정보" -> "실행"을 선택합니다.
- 회사/학교 보안 정책으로 설치가 막히면 zip 파일 방식으로 실행해 봅니다.
- 그래도 실행되지 않으면 오류 화면을 캡처해 개발자에게 전달합니다.

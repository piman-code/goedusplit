Goedu-Split 1.0.1 Windows 설치 및 실행 안내

1. 다운로드

GitHub 저장소로 이동합니다.
https://github.com/piman-code/goedusplit

Releases 메뉴에서 Goedu-Split-1.0.1.zip 파일을 다운로드합니다.

2. 압축 풀기

다운로드한 zip 파일을 오른쪽 클릭한 뒤 "모두 압축 풀기"를 선택합니다.
압축을 푼 Goedu-Split 폴더 전체가 프로그램입니다.

3. 실행

Goedu-Split 폴더 안의 Goedu-Split.exe를 더블클릭합니다.
처음 실행할 때 Windows 보안 안내가 뜨면 "추가 정보"를 누른 뒤 "실행"을 선택합니다.

주의: Goedu-Split.exe만 따로 빼서 실행하지 마세요. _internal 폴더 등 함께 들어 있는 파일이 필요합니다.

4. 예상정답률 탭

예상정답률 입력 탭은 자료를 모두 올리지 않아도 사용할 수 있습니다.
상단의 "전체 예상 분할점수"는 체크한 문항만이 아니라 표 전체 문항 기준입니다.
문항별 배점과 A/B/C/D/E 예상정답률을 모두 합산해서 계산합니다.

5. Codex CLI 클라우드 AI를 처음 쓰는 경우

Codex CLI 방식은 API Key를 입력하지 않습니다.
Windows 터미널 또는 PowerShell에서 ChatGPT 계정으로 한 번 로그인한 뒤 사용합니다.

아래 명령을 순서대로 실행합니다.

winget install OpenJS.NodeJS.LTS
npm install -g @openai/codex
where codex
codex --version
codex login
codex login status

codex login status 결과가 Logged in using ChatGPT로 나오면 준비가 된 것입니다.
그 다음 Goedu-Split에서 AI 문항 검토 > AI 설정 > Codex CLI 클라우드 (OAuth) > 연결 테스트를 누릅니다.

6. 업데이트

새 버전이 올라오면 기존 폴더에 덮어쓰지 말고 새 zip을 내려받아 새 폴더에 압축을 풀어 실행하는 방식을 권장합니다.

7. 보안

배포 zip에는 .git, .env, .venv, 실제 학생자료, API Key, 토큰 파일이 포함되지 않도록 점검했습니다.
AI 문항 검토에서 Codex CLI 클라우드 AI를 사용할 때만 선택한 검토 자료가 Codex CLI를 통해 처리됩니다.

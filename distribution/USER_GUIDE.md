## 무엇을 하는 프로그램인가요?

NEIS에서 받은 **지필평가 정오표**와 학교에서 만든 **문항정보표** 엑셀을 넣으면, 성취수준 분포·문항 난이도·변별도·답지반응분포·성취기준별 도달도까지 한 번에 보여드립니다. **데이터는 본인의 PC를 절대 떠나지 않습니다.**

---

## macOS 사용자 (.dmg 파일을 받으셨다면)

### 설치 방법

1. 받은 `Goedu-Split-x.y.z-mac.dmg` 파일을 더블클릭합니다.
2. 창이 열리면 **Goedu-Split.app** 아이콘을 옆의 **Applications** 폴더 아이콘으로 끌어다 놓습니다.
3. 끝났습니다. 창은 닫고, dmg 파일은 삭제해도 됩니다.

### 실행 방법

⚠️ Mac이 "확인되지 않은 개발자가 만든 앱"이라며 차단할 수 있습니다(개인 개발자 인증서가 없어서 그렇습니다). 다음 중 하나로 한 번만 풀어주면 됩니다.

**방법 A · 가장 쉬운 방법 (권장)**

1. **Finder → 응용 프로그램(Applications)** 으로 갑니다.
2. `Goedu-Split` 아이콘에서 **마우스 우클릭(또는 Control+클릭) → 열기**.
3. 경고창이 다시 떠도 **열기** 버튼을 누릅니다.
4. 한 번 열면 다음부터는 일반 앱처럼 더블클릭으로 열립니다.

**방법 B · 위 방법이 안 될 때**

`터미널` 앱을 열고(Spotlight에서 "터미널" 검색) 아래 한 줄을 복사 후 Enter:

```bash
xattr -dr com.apple.quarantine /Applications/Goedu-Split.app
```

그 후 응용 프로그램에서 더블클릭으로 실행.

### 프로그램 사용 방법

1. 좌측 **학생답 정오표**에서 NEIS에서 받은 정오표 .xlsx를 선택.
2. **문항정보표**에서 학교 양식 .xlsx를 선택.
3. (선택) **예상추정분할점수 조회** 엑셀을 넣으면 분할점수가 자동으로 채워집니다.
4. 큰 파란 **분석 실행** 버튼을 누르면 우측 탭에 결과가 표시됩니다.

⌨️ 단축키: `Ctrl+1` 라이트 모드 / `Ctrl+2` 다크 모드 / `Ctrl+3` 시간대 자동 / `Ctrl+R` 다시 분석 / `Ctrl+E` 결과 CSV 저장.

## Windows 사용자 (.zip 파일을 받으셨다면)

### 설치 방법

1. 받은 `Goedu-Split-x.y.z-windows.zip` 파일에서 **마우스 우클릭 → 압축 풀기** (또는 "모두 압축 풀기").
2. 원하는 폴더(예: `바탕 화면\Goedu-Split`)에 풉니다.
3. 별도 설치 프로그램은 없습니다. 압축을 푼 폴더 전체가 프로그램입니다.

### 실행 방법

폴더 안의 **`Goedu-Split.exe`** 를 더블클릭합니다.

⚠️ 처음 실행 시 *"Microsoft Defender SmartScreen에서 인식할 수 없는 앱…"* 파란색 경고가 뜰 수 있습니다(개인 개발자 서명이 없어서 그렇습니다). 다음 순서로 한 번만 통과하면 됩니다.

1. 경고창에서 **추가 정보** 라는 작은 글씨 클릭.
2. 아래쪽에 나타나는 **실행** 버튼 클릭.

이후부터는 일반 프로그램처럼 더블클릭만으로 열립니다.

### 배포 시 주의

- 선생님들께는 **`Goedu-Split-x.y.z-windows.zip` 파일 하나**만 보내면 됩니다.
- 받는 분은 압축만 풀고 `Goedu-Split.exe`를 실행하면 됩니다.
- 압축을 푼 뒤에는 `Goedu-Split.exe`만 따로 빼서 옮기지 말고, **Goedu-Split 폴더 전체**를 함께 보관해야 합니다.

## Windows 사용자 (setup.exe 설치파일을 받으셨다면)

### 설치 방법

1. 받은 `Goedu-Split-x.y.z-windows-setup.exe` 파일을 더블클릭합니다.
2. Windows SmartScreen 경고가 뜨면 **추가 정보 → 실행**을 누릅니다.
3. 설치 마법사의 안내에 따라 설치합니다.
4. 설치 후 시작 메뉴 또는 바탕 화면 바로가기에서 **Goedu-Split**을 실행합니다.

### 제거 방법

Windows **설정 → 앱 → 설치된 앱**에서 `Goedu-Split`을 찾아 제거하면 됩니다.

### 프로그램 사용 방법

macOS 사용자 안내의 프로그램 사용 방법과 동일합니다.

## 입력 파일 형식 안내

| 파일 | 어디서 받나요? | 핵심 머리글 |
| --- | --- | --- |
| 학생답 정오표 | NEIS → 평가관리 → 지필평가 정오표 다운로드 | `반/번호`, 정답 행, 배점 행, 학생별 응답(`.`이면 정답, 숫자면 학생이 고른 오답) |
| 문항정보표 | 학교에서 출제 시 만든 양식 | `문항번호`, `내용영역`, `성취기준`, `난이도`(어려움/보통/쉬움), `배점`, `정답` |
| 예상추정분할점수 조회 (선택) | NEIS의 추정분할점수 조회 | `A/B`, `B/C`, `C/D`, `D/E`, `E/미도달` 컬럼과 `평균` 행 |

양식이 크게 다르면 인식이 안 될 수 있습니다. 그런 경우 제작자에게 파일 샘플을 보내주세요.

## 자주 묻는 질문

**Q. 데이터가 외부로 나가지 않나요?**
A. 본 프로그램은 인터넷에 접속하지 않습니다. 모든 분석은 PC 내부에서만 이뤄집니다.

**Q. 분석 결과가 KICE 웹앱과 살짝 다른데요?**
A. 본 앱은 KICE 웹앱의 공개 화면을 참고해 같은 산식을 다시 만든 것입니다. Cronbach α 등에서 소수점 셋째 자리 정도 차이가 날 수 있습니다.

**Q. 한글이 □□□로 깨져 보여요.**
A. macOS는 *Apple SD Gothic Neo*, Windows는 *맑은 고딕*이 자동으로 적용됩니다. 안 보인다면 시스템에 한글 글꼴이 설치돼 있는지 확인해 주세요.

**Q. 결과를 엑셀로 저장하고 싶어요.**
A. 메뉴 → 파일 → **결과 CSV 내보내기** (Ctrl+E). 학생결과·문항분석·성취기준별·성취수준분포 4개 CSV가 생성되며 엑셀에서 바로 열 수 있습니다.

**Q. 학교 PC가 너무 느려요.**
A. 처음 실행 시 약 5~15초 로딩이 정상입니다(앱 파일 100~150MB). 두 번째부터는 빨라집니다.
